usb_keyboard_bridge_for_msx を書き込んだ Raspberry Pi Pico の挙動を確認するための
Raspberry Pi Pico プログラムです。（Raspberry Pi Pico が2つ必要です）

Yの端子が出力に、Xの端子が入力に変わっているので、これらを直結して、
期待通りのログが出るか確認します。
