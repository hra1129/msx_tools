CHGCPU

【概要】
FS-A1FX/WX/WSX/ST/GT で、MSX-DOSからCPUモードを切り替えるためのツールです。

【使い方】
MSX-DOS上動作するプログラムです。

A>CHGCPU 番号

番号に下記の値を指定してください。
0 ... Z80 3.579MHzモードへ切り替える (FS-A1FX/WX/WSX/ST/GT)
1 ... R800 ROMモードへ切り替える (FS-A1ST/GT)
2 ... R800 DRAMモードへ切り替える (FS-A1ST/GT)
3 ... Z80 5.369MHzモードへ切り替える (FS-A1FX/WX/WSX)

番号を指定せずに、引数無しで実行すると簡単なヘルプが表示されます。
